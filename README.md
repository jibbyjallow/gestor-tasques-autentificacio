# Gestor de Tasques amb Autenticació

Aquest projecte és una aplicació backend en **Node.js** amb **Express** i **MongoDB** que permet gestionar tasques amb un sistema d’autenticació i roles (`user` i `admin`).

---

## 1. Instal·lació

### Requisits

- Node.js >= 18
- npm
- MongoDB en marxa (local o cloud)

1. Clona el repositori:

git clone https://github.com/jibbyjallow/gestor-tasques-autentificacio.git
